========================================================
   ✨ Integralink Word Add-in 설치 가이드 ✨
========================================================

Integralink를 선택해 주셔서 감사합니다! 🙇‍♂️
이 프로그램은 Integralink Word Add-in을 간편하게 설치할 수 있도록 도와줍니다.

⚠️ [ 필수 요구 사항 ]
- Microsoft Office 2021 이상 또는 Microsoft 365 구독 버전
- (이전 버전에서는 정상적으로 작동하지 않을 수 있습니다)

--------------------------------------------------------
🚀 [ 1. 설치 방법 (Installation) ]

1. 실행 중인 Microsoft Word를 모두 종료해 주세요. ❌
2. 'install.bat' 파일을 마우스 우클릭 후 '관리자 권한으로 실행'을 선택하세요. 🛡️
3. 검은색 창이 뜨고 "Installation Complete!" 메시지가 나올 때까지 기다려 주세요. ✅

--------------------------------------------------------
⚙️ [ 2. 활성화 방법 (최초 1회 필수) ]

설치가 완료된 후, Word에서 한 번만 수동으로 추가해 주시면 됩니다.
이후부터는 홈 탭에 자동으로 나타납니다! 🎉

1. Microsoft Word를 실행하세요. 📘
2. '삽입(Insert)' 탭 (또는 '홈(Home)' 탭의 '추가 기능')으로 이동하세요.
3. '추가 기능 가져오기(Get Add-ins)' 버튼을 클릭하세요. 🧩
4. 창이 뜨면 상단의 **'공유 폴더(SHARED FOLDER)'** 탭을 클릭하세요. 📂
5. 목록에서 **'Integralink'**를 선택하고 '추가(Add)'를 누르세요. ➕

이제 홈(Home) 탭에 Integralink 아이콘이 나타납니다! ✨

--------------------------------------------------------
🗑️ [ 3. 제거 방법 (Uninstallation) ]

프로그램을 삭제하고 싶으시다면:
1. Word를 종료하세요.
2. 'uninstall.bat' 파일을 관리자 권한으로 실행하세요. 🛡️
