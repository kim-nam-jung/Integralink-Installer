========================================================
   🍎 Integralink Word Add-in for Mac 🍎
========================================================

Mac에서는 터미널(Terminal)을 이용해 설치합니다.

⚠️ [ 필수 요구 사항 ]
- Microsoft Word for Mac (최신 버전 권장)

--------------------------------------------------------
🚀 [ 1. 설치 방법 (Installation) ]

1. 터미널(Terminal)을 엽니다. (Command + Space -> Terminal 검색)
2. `cd` 명령어로 이 폴더(`Mac`)로 이동합니다.
   (팁: `cd `를 치고 폴더를 터미널로 끌어다 놓으세요)

3. 다음 명령어를 실행하여 권한을 부여합니다:
   chmod +x install.sh uninstall.sh

4. 설치 스크립트를 실행합니다:
   ./install.sh

5. "Installation Complete!" 메시지가 뜨면 성공! 🎉

--------------------------------------------------------
✅ [ 2. 확인 방법 ]

1. Microsoft Word를 껐다가 다시 켭니다. 🔄
2. 삽입(Insert) > 추가 기능(Add-ins) > 내 추가 기능(My Add-ins)으로 이동합니다.
3. 'Integralink'가 보이면 선택하고 '추가'를 누르세요.

(안 보이면 '개발자 추가 기능' 탭이나 화살표를 눌러 확인하세요)

--------------------------------------------------------
🗑️ [ 3. 제거 방법 (Uninstallation) ]

./uninstall.sh
